# 생각노트

- 주별 LMS 게시판에 올라오는 읽기자료를 바탕으로 작성
- 2페이지 이내
- 해당 주 읽기 자료에 대한 핵심 내용을 종합적으로 요약 (synthesis)
  - synthesis는 summary와는 다름.
  - 병렬적으로 A논문은 ~내용, B논문은 ~내용, 이렇게 쓰는 것이 아니라, 종합적으로 요약해야함
  - synthesis를 1페이지만 쓰고 learning pts를 길게 써도 상관 없음
- learning points와 최소 2개의 질문을 포함할 것
  - 가능하다면 텀페이퍼와 연계될 수 있게
  - 질문은 너무 크게 하지 말것
  - 질문이 정제되고 잘 짜여진 것에 국한되기보다는 거칠고 유니크하더라도 '내 생각들'이 나오는 것을 권고
- LMS 과제란에 화요일 저녁 7시까지 PDF로 업로드
- 경어체 사용 금지
- 생각노트에 한해서는 AI를 활용하지 않는 것을 권고
